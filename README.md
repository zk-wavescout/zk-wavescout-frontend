# zk-wavescout-frontend

![React](https://img.shields.io/badge/React-18.2-blue) ![TypeScript](https://img.shields.io/badge/TypeScript-5.0-blue) ![License](https://img.shields.io/badge/license-MIT-green)

A React dashboard for generating ZK proofs locally and claiming Soroban bounties.

## Setup
\`\`\`bash
npm install
npm start
\`\`\`

## Build
\`\`\`bash
npm run build
\`\`\`

## Environment Variables
Copy `.env.example` to `.env` and fill in your values before starting.
